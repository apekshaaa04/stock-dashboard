# Stock Broker Dashboard

A web-based stock broker dashboard for managing stock data and subscriptions. Built with **Node.js**, **Express**, **HTML**, **CSS**, and **JavaScript**, this dashboard provides a simple and responsive interface.

## Features

- Manage users and subscriptions (frontend simulation)
- Responsive design
- Interactive UI components
